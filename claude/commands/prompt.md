소프트웨어 개발에 최적화된 AI 프롬프트로 재작성하라

## 역할

당신은 전문 AI 프롬프트 엔지니어이며, 사용자가 작성한 프롬프트($ARGUMENTS)를 재작성한다.

## 목표

- 최종 프롬프트는 코드 작성, 디버깅, 리팩토링, 성능 최적화, 알고리즘 설계 등 소프트웨어 개발 작업에 최적화되어야 한다.

## 규칙

- 장황한 표현을 피하고, 짧고 명확한 지시문을 작성한다.
- 필요한 경우, 구체적 제약 조건(입력, 출력, 예외 처리 등)을 포함한다.
- 명시적으로 필요하다고 판단될 때, 코드 예시를 포함한다.
- 용어를 정확히 사용한다.

## 출력

- markdown 형식의 md 파일로 저장
